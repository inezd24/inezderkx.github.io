# Academic Personal Website Static HTML site, ready to drop into your `username.github.io` repo. ## Files - `index.html` — Landing page (hero + recent activity) - `about.html` — Bio, education, appointments, link to CV - `research.html` — Publications list - `projects.html` — Active and past projects - `tools.html` — GitHub packages and Docker images - `contact.html` — Email, office, social links - `assets/styles.css` — All styling - `assets/main.js` — Mobile nav + active link ## Setup 1. Replace `Dr. Your Name` and `[your...]` placeholders throughout the HTML. 2. Replace the two hero/about images (currently `picsum.photos`). 3. Drop your CV at the site root as `cv.pdf` (the About + hero links to it). 4. Update GitHub / Scholar / ORCID / Mastodon URLs in the footer and Contact page. 5. Push to `username.github.io` repo and GitHub Pages serves it automatically. ## Customizing colors Edit the CSS variables at the top of `assets/styles.css`: - `--brand` — primary accent (blue by default) - `--accent` — secondary accent (warm orange) - `--text`, `--text-muted`, `--surface`, `--border` — neutral palette Colors use `oklch()` for perceptual uniformity.
